# TAG Help Centre
Documentation site for the TAG Recruiter Dashboard and TAG Connect worker app.
Built with MkDocs Material. Push to main -> GitHub Action deploys to gh-pages -> served via GitHub Pages.
- Web: browse the site. PDF: open "Print site" (last nav item) and print/save as PDF.
- Screenshots live in docs/assets/screenshots/ - naming: chNN-shortname.png (e.g. ch06-single-review.png)
- Every release: update affected pages + docs/changelog.md, recapture affected screenshots only.
